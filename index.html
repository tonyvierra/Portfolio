Project 1: Autonomous Follower Vehicle (Jetson, Vision + Range Sensing)

Status: In progress
Jetson Vision Range Sensing

Overview

Building an autonomous follower robot around a Jetson Orin Nano that performs vision-based target detection with additional range sensing, fully on-board (no cloud processing).

What I have done

  • Set up and configured the Jetson Orin Nano as the main compute platform
  • Integrated a Pi Camera and a simple LiDAR / range sensor with the Jetson
  • Verified sensor data availability and system bring-up
  • Currently working on the computer vision model selection and real-time inference pipeline

Technical focus

  • Evaluating lightweight CNN-based object detectors (YOLO-style models) for edge inference
  • Designing a perception pipeline that outputs target position and distance in real time
  • Planning control logic that maps image error to steering and distance error to speed

Planned next steps

  • Integrate perception outputs into closed-loop motor control
  • Tune control behavior with respect to camera latency and embedded compute constraints
  • Implement proper power separation between motors and compute during full system integration
